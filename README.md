

# DECO CRM 📱🎉

Application mobile CRM intelligente développée avec **Flutter** et **Firebase**, 
conçue pour DECO PAS PLUS, une entreprise spécialisée dans l'organisation et la 
décoration événementielle (Ezzahra, Ben Arous, Tunisie).

## 🎯 Objectif
Centraliser les données clients, améliorer la segmentation marketing et optimiser 
la performance commerciale de l'activité événementielle.

## ✨ Fonctionnalités principales
- 👥 Gestion Clients (CRM) — profils, historique, source d'acquisition
- 🎊 Module Événementiel — type, date, lieu, budget, pack, thème
- 💰 Devis & Commandes — création, validation, conversion, historique financier
- 📊 Segmentation Clients — par budget, type d'événement, zone, fréquence
- ⭐ Avis & Satisfaction — notes, commentaires, photos
- 💬 Communication Clients — intégration WhatsApp API, notifications
- 📈 Tableau de Bord KPI — clients, prospects, taux de conversion, CA

## 🛠️ Technologies
- **Frontend:** Flutter
- **Backend:** Firebase
- **Base de données:** Firestore
